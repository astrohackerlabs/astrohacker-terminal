# Astrohacker Shell

Astrohacker Shell is the Astrohacker port of Shannon. It is exposed as the
`ahsh` binary and builds on a patched Nushell fork plus upstream Reedline.

This crate is excluded from the default Astrohacker Rust workspace until fork
reconstruction and release packaging are automated. Build it directly from this
directory after preparing `forks/nushell` and `forks/reedline`.
