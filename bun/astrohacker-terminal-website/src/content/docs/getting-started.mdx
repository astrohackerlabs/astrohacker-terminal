<h1>Getting Started</h1>

<p>
  Astrohacker Terminal embeds web browsers inside your terminal. Type
  <code>web localhost:3000</code> and see your work without leaving the shell.
</p>

<h2>Install with Homebrew</h2>

<p>The fastest way to install on macOS:</p>

<pre class="bg-background-dark p-3 border-l-2 border-border text-foreground-dark text-xs mb-6 overflow-x-auto">
  <code>
    <span class="text-primary">brew</span> tap astrohackerlabs/astrohacker
    <span class="text-primary">brew</span> trust astrohackerlabs/astrohacker
    <span class="text-primary">brew</span> install --cask astrohacker-terminal
  </code>
</pre>

<p>
  The Homebrew cask currently installs Astrohacker Terminal <code>0.1.0</code>
  through the <code>astrohacker-terminal</code> cask name. It puts the app at
  <code>/Applications/Astrohacker Terminal.app</code>, the <code>web</code> CLI
  at <code>/opt/homebrew/bin/web</code>, and the Chromium helper under
  <code>/opt/homebrew/opt/astrohacker-terminal-ah-chromiumd/</code>.
</p>

<p>To upgrade to a new version:</p>

<pre class="bg-background-dark p-3 border-l-2 border-border text-foreground-dark text-xs mb-6 overflow-x-auto">
  <code>
    <span class="text-primary">brew</span> update &&{" "}
    <span class="text-primary">brew</span> upgrade --cask astrohacker-terminal
  </code>
</pre>

<h2>Build from Source</h2>

<p>
  For development or if you want to build from source. Requires macOS, the
  <a href="https://www.rust-lang.org/tools/install">Rust toolchain</a>, and a
  Chromium build (~100 GB disk space).
</p>

<h3>Clone</h3>

<pre class="bg-background-dark p-3 border-l-2 border-border text-foreground-dark text-xs mb-6 overflow-x-auto">
  <code>
    <span class="text-primary">git</span> clone{" "}
    <span class="text-success">https://github.com/astrohackerlabs/astrohacker-terminal.git</span>
    <span class="text-primary">cd</span> astrohacker-terminal
  </code>
</pre>

<h2>Build</h2>

<p>
  Build all components (Chromium, Roamium, Ghostboard, and the <code>web</code>
  TUI) with:
</p>

<pre class="bg-background-dark p-3 border-l-2 border-border text-foreground-dark text-xs mb-6 overflow-x-auto">
  <code>
    <span class="text-primary">scripts/build.sh</span> all --release
  </code>
</pre>

<p>Or build components individually:</p>

<pre class="bg-background-dark p-3 border-l-2 border-border text-foreground-dark text-xs mb-6 overflow-x-auto">
  <code>
    <span class="text-primary">scripts/build.sh</span> chromium --release{" "}
    <span class="text-muted"># Chromium + libtermsurf_chromium</span>
    <span class="text-primary">scripts/build.sh</span> roamium --release{" "}
    <span class="text-muted"># Roamium (Chromium engine binary)</span>
    <span class="text-primary">scripts/build.sh</span> ghostboard --release{" "}
    <span class="text-muted"># Astrohacker Terminal GUI</span>
    <span class="text-primary">scripts/build.sh</span> webtui --release{" "}
    <span class="text-muted"># web TUI</span>
  </code>
</pre>

<h2>Install</h2>

<pre class="bg-background-dark p-3 border-l-2 border-border text-foreground-dark text-xs mb-6 overflow-x-auto">
  <code>
    <span class="text-primary">sudo</span> scripts/install.sh all
  </code>
</pre>

<p>This installs:</p>

<ul>
  <li>
    <strong>Astrohacker Terminal</strong> to{" "}
    <code>/Applications/Astrohacker Terminal.app</code>
  </li>
  <li>
    <strong>Chromium helper</strong> + Chromium dylibs to{" "}
    <code>/opt/homebrew/opt/astrohacker-terminal-ah-chromiumd/</code>
  </li>
  <li>
    <strong>web</strong> TUI to <code>/usr/local/bin/web</code> for source
    installs
  </li>
</ul>

<h2>Usage</h2>

<p>Launch Astrohacker Terminal:</p>

<pre class="bg-background-dark p-3 border-l-2 border-border text-foreground-dark text-xs mb-6 overflow-x-auto">
  <code>
    <span class="text-primary">open</span>{" "}
    <span class="text-success">"/Applications/Astrohacker Terminal.app"</span>
  </code>
</pre>

<p>In any Astrohacker Terminal pane, open a browser:</p>

<pre class="bg-background-dark p-3 border-l-2 border-border text-foreground-dark text-xs mb-6 overflow-x-auto">
  <code>
    <span class="text-primary">web</span> ryanxcharles.com
  </code>
</pre>

<p>
  The browser renders inline, in the same pane. Use <code>Esc</code> to switch
  from Browse mode to Control mode. Use <code>Enter</code> to switch back. See
  <a href="/docs/components/webtui">Web TUI</a> for the full keybinding
  reference.
</p>

<h2>After macOS Updates</h2>

<p>
  After a macOS update, rebuild Chromium. The Seatbelt sandbox APIs are tied to
  the macOS SDK version — a stale build will crash child processes.
</p>

<pre class="bg-background-dark p-3 border-l-2 border-border text-foreground-dark text-xs mb-6 overflow-x-auto">
  <code>
    <span class="text-primary">scripts/build.sh</span> chromium --clean
    <span class="text-primary">scripts/build.sh</span> all --release
    <span class="text-primary">sudo</span> scripts/install.sh all
  </code>
</pre>
